# College Admission System (Java Swing)

Ye ek simple aur effective **Admission Management System** hai jo Java aur MySQL ke zariye banaya gaya hai.

## 🚀 Features
* **User-Friendly Interface:** Java Swing (GUI) ka istemal kiya gaya hai.
* **Database Integration:** MySQL ke saath connect hai taake data save ho sake.
* **Form Validation:** Name, Father Name, aur Email wagera enter karne ke liye fields.

## 🛠️ Tech Stack
* **Language:** Java
* **GUI Library:** Swing / AWT
* **Database:** MySQL (JDBC Connector)
