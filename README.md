# College Admission System
**Developed by: MUHAMMAD JAWAD & ROHAIL AHMAD**

A professional desktop application built with **Java Swing** and **MySQL** to manage student admissions efficiently.

## 🚀 Key Features
* **Interactive GUI:** Developed using Java Swing and AWT for a smooth user experience.
* **Database Management:** Integrated with MySQL for secure data storage and retrieval.
* **Form Validation:** Real-time checking of input fields like Name, Father Name, and Email.
* **Modern Layout:** Organized components with custom bounds for a clean interface.

## 🛠️ Technical Stack
* **Language:** Java (JDK 8 or higher)
* **Frontend:** Java Swing, AWT
* **Backend:** MySQL
* **Driver:** JDBC MySQL Connector
